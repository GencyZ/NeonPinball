## 推荐推进顺序（P1 先）

**P1（地基 + 便宜高回报）**：连击计分 📋 → 目标钉 📝 → 实时分数 ticker → 推板 Nudge
**P2（锦上添花）**：霓虹边框 📋 → 蓄力发射 → Boss 预告+配额进度条 → 漏斗接球免费球
**P3（大子系统 / 后期）**：特殊球种 → 棋盘多样性 → 豪赌机制 → cascade → Balatro tally → run map → 美术资源 → 全屏

> 顺序非硬性；每项动手前单独 brainstorm→spec→plan。数值/范围以各自 spec 为准。
